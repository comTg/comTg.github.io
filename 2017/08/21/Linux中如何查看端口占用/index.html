Ubuntu中如何查看端口占用

By Vetoer at: 2017-08-21
linux ubuntu 命令
在程序运行过程中，经常会因为一些不当的操作导致应用不能正常关闭，依然占用着端口，影响其他应用的运行

  1. 查看所有的服务端口

  2. 查看所有的服务端口，显示pid号

    netstat -a

  3. 查看指定端口信息共有四种方式(以8000为例):

    ​

    1. 查看某一个指定端口，则可以结合grep(|)命令使用，过滤出当前端口

      netstat -ap | grep 8000

    2. 查看指定端口也有另一种方式，在终端中输入

      lsof -i:8000

    3. 使用netstat -lnp命令查询

      sudo netstat -lnp | grep 8000

    4. 查看端口号和运行程序

      netstat -atunp | more

    5. 查看进程所用端口

      netstat -tlnp | grep 8000

  4. 使用kill杀死对应pid的进程

​ kill pid
