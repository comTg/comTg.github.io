Ubuntu生存指南

By Vetoer at: 2017-08-25
切换开关机动画

sudo update-alternatives --config default.plymouth

#####增加ubuntu对python3中tkinter的支持

sudo apt-get install python3-tk

安装快照 snapd

sudo apt install snapd

使软件开机启动

生成该软件的桌面图标,然后把桌面图标拖到 /etc/xdg/autostart 里面即可
