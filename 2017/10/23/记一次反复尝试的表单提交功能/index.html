记一次反复尝试的禁止刷新页面表单提交功能

Java SpringMVC 2017/10/23

先说遇到的问题

  1. 在还没写js验证表单提交数据的时候,用户点击登录后提交空数据,需要在后端进行验证,增加了大段代码.
  2. js验证表单后,用户数据通过验证进入controller验证的时候,因为在控制层中如果用户输入数据不正确是转发回login页面,所以当用户刷新的时候会出现跳过js重复验证的情况.
